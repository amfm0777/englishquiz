Arquitectura Core del Juego

Desglose de los ficheros esenciales que componen la lógica, diseño y estructura visual de English Quiz. Esta vista ignora configuraciones de entorno y se centra exclusivamente en el código útil de la aplicación.

🏗️ www/index.html
Estructura UI Vistas

El esqueleto principal de la aplicación. Define el layout completo bajo una arquitectura de Single Page Application (SPA).

  • Contenedores de Pantalla: Define todos los div principales que actúan como "pantallas" (`screen-home`, `screen-game`, `screen-dictionary`, `screen-phonemes`, `screen-admin`).
  • Estructura DOM: Plantillas para los botones flotantes, modales de feedback (acierto/error), paneles laterales y animaciones base.
  • Importación: Punto de entrada único donde se cargan los estilos CSS y los scripts JS.
⚙️ www/app.js
Lógica de Juego Estado Global

El motor cognitivo del juego. Contiene toda la lógica de estado y las recciones a los eventos del usuario.

  • State Management: Maneja variables globales como `gameState` (preguntas, índice actual, puntuación) y `currentGameMode` (palabras o fonemas).
  • Navegación: Función `showScreen()` que gestiona la visibilidad de las distintas pantallas (activando y desactivando clases CSS).
  • Lógica de Quiz: Controla el bucle principal de las partidas: seleccionar preguntas con `shuffle()`, inyectarlas al DOM, verificar si son correctas y manejar la pantalla final de resultados.
  • Panel de Control: Configuración e interfaz para que el padre administre y añada nuevas palabras utilizando la API de MyMemory para las traducciones automáticas.
  • Text-To-Speech (TTS): Integración con la voz nativa de Capacitor y un fallback de Web Speech API para leer las palabras en inglés (`speakWord()`).
🎨 www/style.css
Diseño Animaciones

Hoja de estilos responsable del look & feel avanzado y moderno de la aplicación (modo oscuro azulado).

  • Sistema de Diseño: Centraliza colores y dimensiones utilizando variables CSS en `:root`.
  • Responsive Flexbox/Grid: Gestiona la adaptación de las pantallas, el menú, y la lista del diccionario para todos los tamaños de dispositivos móviles.
  • Micro-interacciones: Define las transiciones CSS al pasar el ratón (hover) de botones y las impresionantes animaciones keyframe como `shake` (error), `pop` (acierto) y efectos de aura (`pulse-speak` y circulos SVG).
📚 www/dictionary.js
Base de Datos Estática

La fuente inicial de la verdad (Seed Data) para el vocabulario de la app, que luego se fusiona en `app.js` con las palabras creadas por el usuario en localStorage.

  • Constante Global: Exporta una matriz de objetos con IDs únicos llamada `DICTIONARY`.
  • Estructura de Datos: Cada palabra incluye las claves `en` (Inglés), `es` (Español) y `ph` (Transcripción Fonética).
🔧 www/utils.js
Helpers genéricos

Pequeña librería de funciones utilitarias que no dependen del estado concreto del juego.

  • Algoritmos Reutilizables: Contiene funciones matemáticas puras, esencialmente enfocadas en como barajar los arrays de opciones para el Quiz usando un algoritmo seguro tipo Fisher-Yates (`shuffle()`).
