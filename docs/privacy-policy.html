Política de Privacidad

Última actualización: 4 de marzo de 2026

1. Información General

EnglishQuiz es una aplicación educativa gratuita diseñada para aprender vocabulario inglés-español. Esta política describe cómo gestionamos la información del usuario.

2. Datos que Recopilamos

EnglishQuiz no recopila datos personales. La aplicación no requiere registro, cuenta de usuario, ni inicio de sesión.

Los únicos datos almacenados son:

  • Mejor puntuación — guardada localmente en el dispositivo
  • Palabras añadidas por el usuario — guardadas localmente en el dispositivo
  • Contraseña del panel de padre — guardada localmente en el dispositivo

Todos estos datos se almacenan exclusivamente en el dispositivo del usuario mediante localStorage y nunca se envían a servidores externos.

3. Servicios de Terceros

La aplicación utiliza los siguientes servicios de terceros:

  • Google AdMob — para mostrar anuncios. AdMob puede recopilar información del dispositivo según su propia política de privacidad.
  • MyMemory Translation API — para traducción automática de palabras. Se envía únicamente la palabra a traducir, sin datos personales.
  • Free Dictionary API — para obtener la fonética de las palabras. Se envía únicamente la palabra consultada.

4. Publicidad

La aplicación muestra anuncios mediante Google AdMob. Google puede utilizar identificadores del dispositivo y cookies para personalizar los anuncios. El usuario puede gestionar sus preferencias de anuncios en la configuración de anuncios de Google.

5. Datos de Menores

Esta aplicación está diseñada como herramienta educativa y puede ser utilizada por menores bajo supervisión parental. No recopilamos intencionadamente información personal de menores de 13 años.

6. Seguridad

Toda la información se almacena localmente en el dispositivo. No mantenemos servidores ni bases de datos con información de usuarios.

7. Cambios

Nos reservamos el derecho de actualizar esta política. Los cambios se publicarán en esta misma página.

8. Contacto
